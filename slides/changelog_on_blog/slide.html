gitコマンドでもチェンジログのrawが生成できます。 git log --date=short --pretty=format:"%ad %an %s (#%h)" -10

Format

書式はプロジェクトで統一されていれば、特に規定はありません。

  • Entry - 日付で変更を記録します。
  • Item - 変更の概要を記載します。

Entry Example

YYYY-MM-DD  Foo Bar  <foobar@example.com>

  * README.md: Fixed installation
  * README.md: Added new a File
  * Initial commit
Jekyllを利用してチェンジログを書くとentryごとにファイルを分けることができます。

Item English

英語でチェンジログはだいたいAddedかFixedで始まります。

英語 過去形 意味
Add Added 追加
Fix Fixed 修正
Change Changed 変更
Remove Removed 削除
Rename Renamed リネーム
Refactor Refactored リファクタリング
